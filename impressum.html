Datenschutz

Auf dieser Website besteht keine eigene Benutzerregistrierung. Das Projektteam speichert keine über das eingebettete Petitionsformular eingegebenen Personendaten.

Online-Petition über openPetition

Die Online-Petition wird über ein eingebettetes Angebot von openPetition bereitgestellt. Angaben, die im Petitionsformular eingegeben werden, werden direkt durch openPetition verarbeitet und nicht durch das Projektteam auf dieser Website gespeichert.

Beim Aufruf des eingebetteten Petitionsformulars können technische Verbindungsdaten, insbesondere die IP-Adresse sowie Informationen zum verwendeten Browser und Endgerät, an openPetition übermittelt werden. Für diese Datenbearbeitung gelten die Datenschutzbestimmungen von openPetition.

Bereitstellung der Website

Beim Aufruf dieser Website können durch den Hostinganbieter technisch erforderliche Verbindungs- und Protokolldaten verarbeitet werden. Dazu können insbesondere die IP-Adresse, Datum und Uhrzeit des Zugriffs, aufgerufene Dateien sowie Angaben zum Browser gehören.

Schriftarten und eingebettete Inhalte

Auf dieser Website werden Schriftarten und teilweise Inhalte externer Anbieter eingebunden. Beim Laden solcher Inhalte kann eine direkte Verbindung zu den Servern des jeweiligen Anbieters hergestellt werden. Dabei können technische Verbindungsdaten übermittelt werden.
